## 💻 How to Run a Project

1. Clone the repo:  
```bash
git clone https://github.com/AliAbdullahpgr/React-Practice.git
````

2. Navigate to a project folder:

```bash
cd 02counter
```

3. Install dependencies:

```bash
npm install
```

4. Start the project:

```bash
npm run dev
```

> Note: Most projects use **Vite** as the React boilerplate.
